Стартовые проекты для турнира Develobear 2018.


Ниже представлены инструкции по открытию и сборке стартовых проектов.


====
Java
====

1. Запустить IntelliJ IDEA с рабочего стола.

2. Open project, выбрать game_java.

3. Build -> Build project.

4. Build -> Build artifacts.

5. Отправлять на сервер нужно jar-файл.

=====
C/C++
=====


1. Открыть Solution из директории game_c или game_cpp.

2. Выполнить сборку проекта.

3. Отправлять на сервер нужно исполняемый файл созданный в режиме Release.
   
Приложение собранное в режиме Debug на сервере работать не будет.


===========
Free Pascal
===========


1. Запустить Free Pascal IDE с рабочего стола.

2. File -> Change dir..., выбрать директорию game_fpc, OK.

3. File -> Open, выбрать .pas файл, OK.

4. Compile -> Build.

==========
Pascal ABC
==========


1. Файл -> Открыть, выбрать директорию game_abc, выбрать .pas файл, OK.

2. Программа -> Компилировать.
